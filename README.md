# Host Cloud
Webapp made to learn MVC, Entity Framework and database transactions.

## Used technologies
- ASPNET Core 5 MVC
- Bootstrap
- JQuery
- EntityFrameworkCore 5
- ASPNET Core 5 Identity

## Requirements
WhoisXMLAPI Key - https://main.whoisxmlapi.com/
